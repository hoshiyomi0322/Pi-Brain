# 出境流程
1. ### 提早3小時到機場
2. ### Airport check-in、搭乗手続き(とうじょうてつづき)、チェックイン、機場報到
    - 出示Passport + Ticket → 換取Boarding pass
    - 地點
        |櫃檯辦理登機|自助報到 (self check-in)|
        |:---:|:---:|
        |航空公司的報到櫃檯|自助報到機|
        - 從航班表查詢櫃檯位置
    - 時間：起飛前2小時
3. ### Checked baggage、預け入れ手荷物(あずけいれてにもつ)、受託手荷物(じゅたくてにもつ)、托運行李
    - 出示Passport + Boarding pass
4. ### Security check、保安検査場(ほあんけんさじょう)、安全檢查
    1. 倒水
    2. 掃描Boarding pass
    3. 放置隨身行李
5. ### Passport control、パスポートコントロール、證照查驗
6. ### Boarding gate、搭乗口(とうじょうぐち)、搭乗ゲート(とうじょうゲート)、登機門
    - 出示Passport + Boarding pass

# 入境流程
1. ### Immigration、入国審査(にゅうこくしんさ)
    - #### 出示Passport + QR code/入境表單
2. ### Baggage reclaim、手荷物受取所(てにもつうけとりじょ)
3. ### Customs、税関(ぜいかん)
    - #### 出示Passport + QR code/海關申報單
