# Command
- コマ投げ：→↓←↑手
    - Air コマ投げ：8 + →↓←↑腳
- 旋風手：手手
    - 前旋風手：手手 + 前
    - 後旋風手：手手 + 後
    - OD旋風手：PPP
        - 前OD旋風手：PPP + 前
        - 後OD旋風手：PPP + 後
- 西伯利亞：→↓←手(Far)
    - 俄式背投げ：→↓←手(Close)
- 抓腳：22K
