# Expected Value (Expectation, Mean)
- ### $`E\left[X\right]=\sum\limits_{i=1}^{n}{x_ip_i}=\begin{cases}{\int_{-\infty}^{\infty}{xf(x)\,dx}}&\text{if }X\text{ is Continuous}\\{\sum\limits_{i=1}^{n}{(x_i\cdot f(x_i))}}&\text{if }X\text{ is Discrete}\end{cases}`$
    - $`p_i=\text{Probability of }x_i`$
    - $`f(x)`$ = [Probability Function](distribution-function.md#probability-function)
- ### Properties
    - $E\left[c\right]=c$
    - $E\left[aX+b\right]=aE\left[X\right]+b$
# Expected Value of a Multiple Random Variable
- ### $E\left[X+Y\right]=E\left[X\right]+E\left[Y\right]$
- ### $E\left[XY\right]=E\left[X\right]E\left[Y\right]$
