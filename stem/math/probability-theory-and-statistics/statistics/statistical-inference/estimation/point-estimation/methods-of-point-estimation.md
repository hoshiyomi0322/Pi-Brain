# Maximum Likelihood Estimation (MLE)
- ### Likelihood Function：$`L\left(θ|X\right) = \prod\limits_{i=1}^n{P\left(X=x_i|θ\right)}`$
    - #### Log-Likelihood Function：$`\ln{L\left(θ|X\right)} = \sum\limits_{i=1}^n{\ln{P\left(x_i|θ\right)}}`$
- ### [Maximum](../../../../../algebra/calculus/differential-calculus.md#extremum)：$`\hat{θ} = \argmax\limits_θ{L\left(θ|X\right)},~\hat{θ} = \argmax\limits_θ{\ln{L\left(θ|X\right)}}`$
    - ### $`\big( θ=\hat{θ} ,~ L\left(θ|X\right)=\text{Maximum} \big) ,~\text{when } \big( \frac{d}{dθ}\ln{L\left(θ|X\right)} = 0 ,~ \frac{d^2}{dθ^2}\ln{L\left(θ|X\right)} < 0 \big)`$
