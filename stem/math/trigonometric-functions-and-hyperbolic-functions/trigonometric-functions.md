# Half-Angle Formulas
- ### Derived from [Double-Angle Formulas ($`\cos{2őł}`$)](#double-cos)
- ### $`\sin{\frac{őł}{2}} = \pm\sqrt{\frac{1-\cos{őł}}{2}}`$
