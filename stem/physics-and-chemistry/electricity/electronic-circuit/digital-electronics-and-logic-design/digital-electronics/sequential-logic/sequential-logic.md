# State Encoding
|Binary|Gray Code|One-Hot|
|:---:|:---:|:---:|
|000|000|00000001|
|001|001|00000010|
|010|011|00000100|
|011|010|00001000|
|100|110|00010000|
|101|111|00100000|
|110|101|01000000|
|111|100|10000000|
- ### Binary Encoding
- ### Gray Code：two continuous values differ in only one bit
- ### One-Hot：only one of the bits of the value is $1$
