# Path
```bash
/ # Root Directory or Path Separator
./ # Current Directory
```
- ### n Levels up：`../`*n
    ```bash
    ../ # Parent Directory (1 level up)
    ../../ # 2 levels up
    ../../.. # 3 levels up
    ```

