# Initialize

```bash
git init # initialize

```
# Workflow
```bash
git status # 查看目前檔案狀態

git add . # 將所有變更加入暫存區 (Stage)
git add file.txt #僅加入特定檔案
git add -u

git commit -a
git commit -m "message" # 提交變更
git diff # difference, 查看工作目錄與暫存區的差異

git restore
git reset
```
