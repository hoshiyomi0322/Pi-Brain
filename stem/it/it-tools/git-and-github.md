# Git Flow
- ### Main Branch
    - Master
    - Develop
- ### Support Branch
    - Hotfix
    - Rrelease
    - Feature
---
- ### Repository(repo)
- ### add → commit → push
- ### fetch → merge → pull

# Git Command
```bash
git init
git clone
```
```bash
git status #查看目前檔案狀態 (最常用！)

git add . #將所有變更加入暫存區 (Stage)
git add file.txt #僅加入特定檔案
git add -u

git commit -a
git commit -m "message" #提交變更
git diff #查看工作目錄與暫存區的差異
git push

git fetch
git merge
git pull

git restore
git remote
git reset

git branch
```
