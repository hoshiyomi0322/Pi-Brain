# Shortcuts
- ### Multiple Cursors
    |Shortcuts|Function|
    |:---:|:---:|
    |ctrl+alt+↑/↓|create cursor|

- ### Line
    - `alt+↑/↓`：moving line
    - `alt+shift+↑/↓`：duplicating line

- ### Line
    |Shortcuts|Function|
    |:---:|:---:|
    |`alt+↑/↓`|moving line|
    |`alt+shift+↑/↓`|duplicating line|
