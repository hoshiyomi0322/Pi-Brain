# HTTP Status Code
- ### 1xx Informational Response
  - 100 Continue
  - 101 Switching Protocols
- ### 2xx Success
    - 200 OK
    - 201 Created
    - 202 Accepted
    - 204 No Content
- ### 3xx Redirection
    - 304 Not Modified
- ### 4xx Client Error
    - 403 Forbidden
    - 404 Not Found
    - 405 Method Not Allowed
    - 406 Not Acceptable
- ### 5xx Server Error
