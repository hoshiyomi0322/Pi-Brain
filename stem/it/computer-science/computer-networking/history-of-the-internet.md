# History of the Internet
- ### 1969：Advanced Research Projects Agency Network (ARPANET)
    - ### Larry Roberts：created the ARPANET, Father of the ARPANET
    - ### the first packet-switched network
- ### 1971：the first Electronic mail (email)
    - ### Use the '@' symbol to separate the Username from the Host
- ### 1973：[TCP/IP Protocol Suite](./communication-protocol/communication-protocol.md#tcpip-protocol-suite-internet-protocol-suite-dod-model)
    - ### Vint Cerf, Robert Kahn：created the TCP/IP Protocol Suite, Father of the Internet
- ### 1983：[Domain Name System (DNS)](./communication-protocol/protocol-layer/dns.md)
- ### 1984：[OSI model](./communication-protocol/communication-protocol.md#open-system-interconnection-model-osi-model)
- ### [Tim Berners-Lee](#tim-berners-lee-1)
- ### 1990：Archie, the first Search Engine
- ### 1993：NCSA Mosaic, the first Browser to display images inline with text
- ### 1994：[HTTP Secure (HTTPS)](./communication-protocol/protocol-layer/protocol-layer.md#http-secure-https)
- ### 1998：Google
- ### 2004：Facebook
- ### 2005：YouTube
- ### 2006：Twitter

# Tim Berners-Lee
- ### 1989：[HyperText Transfer Protoco (HTTP)](./communication-protocol/protocol-layer/http.md)
- ### 1990
    - ### WorldWideWeb (Nexus), the first Web Browser
    - ### CERN httpd, the first web server
    - ### [Uniform Resource Identifier (URI)](computer-networking.md#uniform-resource-identifier-uri), [Uniform Resource Locator (URL)](computer-networking.md#uniform-resource-locator-url)
- ### 1991：[World Wide Web (WWW)](computer-networking.md#world-wide-web-www)
- ### 1993：[HyperText Markup Language (HTML)](../../coding/markup-language/html/html.md)
